### Description :

A full-stack web application that helps students to find their best scholarship. Designed to bridge the gap between students and opportunities, this platform uses machine learning to recommend scholarships tailored to each student's unique profile â€” making the search process smarter, faster, and more accessible.


### Key Features:

- Personalized scholarship recommendations using a Decision Tree Classifier

- Profile-based search filtering (qualification, income, category, location)

- Chatbot for intuitive scholarship search using fuzzy matching

- Secure user authentication and profile management

- Structured data management with MySQL

### Tech Stack:

- Frontend: React.js, Tailwind CSS
- Backend: FastAPI (Python), JWT Authentication
- Machine Learning: Scikit-learn (Decision Tree Classifier), Pandas, Pickle
- Database: MySQL
- Other: FuzzyWuzzy (chatbot fuzzy matching)
